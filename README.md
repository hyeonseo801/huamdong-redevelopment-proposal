# 후암동 역세권 완전개발 제안
 
서울 최고 입지의 방치된 가능성 — 데이터가 놓친 것을 현장에서 찾다
 
## 배경
 
[Gen-City](https://github.com/hyeonseo801/Gen-City)를 개발하면서 한 가지 한계를 마주쳤다.  
교통 접근성, 공시지가, 의료시설 거리 — 정량 지표들은 입지를 잘 설명하지만,  
왜 어떤 땅이 그 입지에 비해 개발되지 않았는지는 설명하지 못했다.
 
2023년 후반, 직접 후암동을 찾아갔다.  
숙대입구역에서 나와 골목으로 들어서는 순간, 서울역이 도보 15분 거리라는 사실이 믿기지 않았다.  
Gen-City 입지 분석 점수는 68점 언저리. 나쁘지 않은 숫자다.  
그런데 현장은 그 점수와 전혀 다른 얼굴을 하고 있었다.
 
파고들수록 문제가 단순하지 않았다 — 고도제한이라는 제도적 억제,  
오래된 주민 커뮤니티와 골목 문화, 젠트리피케이션에 대한 우려가 뒤엉켜 있었다.  
개발이 안 된 게 아니라, 개발이 쉽지 않은 이유가 겹겹이 쌓여 있던 거다.
 
2024년 고도제한이 완화됐다. 억제의 조건 중 하나가 바뀐 지금,  
나머지 문제들을 어떻게 풀어낼 수 있는지를 제안하는 게 이 문서의 목적이다.

## 관련 프로젝트
 
이 제안서는 [Gen-City](https://github.com/hyeonseo801/Gen-City) 개발 과정에서 발견한 문제의식을 바탕으로 작성됐다.
